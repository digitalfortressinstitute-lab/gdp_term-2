# 🌐 Module 7 — RESTful APIs

> **GDP Term 2 | Digital Fortress Institute**
> Diploma in Full Stack Software Development | Year 1, Term 2

---

## 📌 Module Overview

This module extends what you learned in **Module 6 (Spring Boot)** and takes you into production-grade API development. You will learn how to design, implement, secure, and test REST APIs using **Java Spring Boot**.

---

## 📚 What You Will Learn

| Topic | Description |
|---|---|
| REST Design Principles | Richardson Maturity Model, HTTP verbs, status codes |
| API Design | DTO pattern, validation, versioning, pagination |
| Middleware | Filters, interceptors, exception handling |
| Security | API key auth, CORS, rate limiting, input sanitisation |
